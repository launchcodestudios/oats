Fade in/out

To fade an element in, start with both classes oats-fadeout oats-fadein on the element.
Remove the oats-fadeout class to make it fade in.
Add back the oats-fadeout class to make it fade out.

<div class="oats-fadein oats-fadeout">Something to fade in and out</div>
